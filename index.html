This library boils down to providing HTTP producers (e.g. HTTP standalone servers) and concepts to permit user-provided producers to collaborate (i.e. communicate) with any other written HTTP consumer. A few HTTP consumers (e.g. file server) are also provided to increase your productivity. It's fine to use this library to write a web chat application.

This library is not about writing template-driven web pages, but it is about HTTP. You should be able to do things like video live streaming via HTTP (try to do that with old school web frameworks).
